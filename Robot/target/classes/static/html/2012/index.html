第十二届中国青少年机器人竞赛

中国青少年机器人竞赛

中国青少年机器人竞赛活动是中国科协主办的面向21世纪青少年的系列科技创新活动之一。该活动始办于2001年，竞赛旨在以丰富多彩、形式多样的机器人探 究项目，培养青少年的创新精神和动手实践能力，激发他们对科学技术以及机器人研究应用的兴趣，提高他们的科学素质。
