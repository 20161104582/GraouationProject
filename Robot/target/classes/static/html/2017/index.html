机器人综合技能比赛--全民健身

机器人综合技能比赛面向中小学生，在固定的场地上设置数个不同难度的任务。机器人综合技能比赛可以检验青少年对机器人技术的理解、对机械结构的认识以及对基本程序编写的掌握程度，可以激发青少年对机器人技术的兴趣，培养他们动手、动脑的能力。经过比赛，学生们不仅完成自己的比赛机器人，也提升了对科技和利用科技来积极影响周围世界的认识。本届机器人综合技能比赛主题为“全民健身”。

WER工程创新赛--工业时代

VEX机器人工程挑战赛--星光璀璨

VEX机器人工程挑战赛也是一项国际青少年机器人比赛项目，每年设置一个主题并设计新的竞技内容。比赛采用联队对抗形式进行，参赛青少年通过遥控器控制己方的两个机器人获得尽量多的分数，同时还要合理运用战略防止对方得分。VEX比赛对抗性强、展示性高，深受青少年的喜爱。

本届VEX挑战赛主题是“星光璀璨”，比赛采用联队对抗形式进行，分手动控制和自动控制两个阶段，以此来培养青少年沟通协调、团队合作和创造性解决问题的能力。

FLL机器人工程挑战赛--动物之友

FLL机器人工程挑战赛是一项青少年国际机器人比赛项目，整个竞赛分为现场竞技、技术问辩、课题研究。比赛每年设置一个主题，围绕主题设计十余项任务，由机器人在固定场地内完成，同时要求参赛团队结合生活对这个主题开展课题研究，最后在技术问辩时进行课题介绍和展示团队风采，这也是对参加活动青少年的一次综合、全面的考察。

本届FLL机器人工程挑战赛的主题是“动物之友”，旨在引导学生关注动物保护，了解人类与动物相互依存的关系，并学习掌握动物学、环境保护的知识。

机器人创意比赛--我的学习伙伴

机器人创意比赛是一项自由度比较高的竞赛。学生在辅导老师的指导下，在学校、家庭、校外机器人工作室或科技实验室里，以个人或小组的方式进行智能机器人的创意、设计、编程与制作，最后完成自己的机器人创意作品参加竞赛展示和评选，整个制作过程可以持续6个月左右。是一项充分发挥青少年创新性思维，培养团队协作能力，展现他们综合素质的竞赛项目。

本届机器人创意比赛的主题是“我的学习伙伴”，旨在促进学生了解人类学习方式的转变、新技术如何辅助人类学习，提高教育质量，同时尝试创意一个改善或辅助人学习的机器人作品。
