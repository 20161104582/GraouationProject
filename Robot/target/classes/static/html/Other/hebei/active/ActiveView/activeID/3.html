安徽青少年科技创新大赛

安徽青少年科技创新大赛是安徽中、小学各类科技活动优秀成果集中展示的一种形式，其目的在于激发青少年科学兴趣，培养和提升我省青少年的创新创造精神和实践探索能力。安徽青少年科技创新大赛经过多年不断发展和完善，在活动内容、活动形式等各方面不断汲取省内外成功的经验已成为省内面向学校中小学生开展的规模最大、层次最高的青少年科技教育活动。

认识生物进化的学校课堂飞行器

认识生物进化的学校课堂飞行器

作者：阚宇成，作者学历：小学五年级，作者学校：合肥实验学校，年龄：10岁，性别：男，辅导教师：戴韵茹，项目简介：我们现在的课堂上需要学习很多科普知识，比如远古时期的动植物生长，比如现代科技中的基因破译，可是这些生物知识都离我们太遥远，理解起来特别费劲，记忆起来也特别麻烦。如果能让我们在学习这些枯燥知识的时候，有一艘飞船带着我们去穿越，到史前时期、到原始森林中、到浩瀚的海洋里等等，让我们能亲眼见证这些动植物的生长环境和它们的生活习性，那么我们的学习就会变得非常有意义，充满乐趣。

沙漠加工器

沙漠加工器

作者：张子琪，作者学历：初中一年级，作者学校：濉溪县任集中心学校，年龄：13岁，性别：女，辅导教师：王伟伟，项目简介：我的作品是这样设计的：画的下面是一片广阔的沙漠，我们要保护水源，植树造林，让阳光更加灿烂，使空气愈加清新，让天空更加蔚蓝，鸟声啁啾，流水淙淙，松涛阵阵， 雨雪霏霏。 我想让沙漠都变成了绿洲， 浩瀚沙漠就披上了从没有穿过的 绿衣服，无边的绿色衣服。所以我设计了一个让沙漠长出树的机器，这个机器能放出紫外线让沙漠长出树来，我真希望在不久的将来，我的梦想变成了现实，那有多好 啊！！

复合多功能转换机器虫

复合多功能转换机器虫

作者：李筱迈，作者学历：小学三年级，作者学校：合肥市蜀麓小学，年龄：9岁，性别：女，辅导教师：张丽，项目简介：随着科技的发展，农业技术不断进步，未来农药除草剂化学肥料等在农作物生长过程中将不在使用。我设计的这款复合多功能转换机器虫自带太阳能，光能收集系统，两只眼睛在夜晚能发出吸引害虫的有色光，白天能散发吸引害虫的特殊气味，因此不管是水里的农作物还是陆地的农作物，不管是白天还是黑夜，机器虫都能把害虫及时收集处理掉，并将害虫转化为新的生物肥料，回归到土壤里，为农作物的生长提供更好的养料，这款机器虫的诞生为我们将来食品安全提供保障。
