公告栏

各市州、直管市、神农架林区青少年科技教育机构：

    太空种子种植活动作为我省青少年科学调查体验活动拓展项目持续开展以来，一直深受基层老师和学生们的喜爱。为鼓励和培养青少年的动手能力、观察能力、思考能力，在实践中学习科学知识、体验科学方法、培养科学兴趣，2020年湖北省青少年科技中心将继续开展此项活动，现将有关事项通知如下：

    一、活动名额

    2020年太空种子种植活动单位50个，统一配发太空种子和《活动指导手册》。名额分配如下：

    1.保留10所2019年度活动扎实并成果显著的学校（具体名单见附件1）；

    2.各市州重新推荐2-3所符合条件的活动学校，湖北省青少年科技中心将从中遴选40所。

    二、活动要求

    1.申报条件：具备种植场地、有专人负责并积极组织学生开展种植活动的中小学校（同等条件下优先考虑青少年科学调查体验活动推广示范学校）均可申报活动单位。未积极开展活动、不按要求报送活动资料的往年活动单位不具备申报资格。

    2.活动要求：活动单位积极组织学生开展种植活动，可结合本地实际情况，组织开展相关拓展活动，如对比种植、科学育种、科学摄影、科学小论文、小小农学家等多形式、多内容的种植探究活动，并按要求定期向省青少年科技中心提交活动资料，内容可以是活动简讯、活动日志、记录表、照片、视频等。

    三、活动程序

    1.申报阶段：各市州主管单位于2月14日前，将《太空种子种植活动单位推荐表》（附件2）盖章后报送湖北省青少年科技中心，并汇总本地区活动单位2020年太空种子种植活动方案报至湖北省青少年科技中心邮箱。

    2.活动阶段：活动单位组织学生开展太空种子种植活动，安排辅导教师、邀请农技专家进行专门辅导，指导学生对种子种植的全过程进行详细记录。同时，结合本地实际情况，开发设计内容丰富、形式新颖的拓展活动。

    3.展示阶段：6-10月，活动单位应及时将活动过程和成果上传至QQ工作平台，进行交流分享。活动结束后，将活动总结及相关拓展活动成果，于11月30日前报至湖北省青少年科技中心指定邮箱。

    四、联系方式

    湖北省青少年科技中心科普活动部

各相关单位：

    根据《中国科协办公厅 教育部办公厅关于开展2020年“英才计划”工作的通知》（科协办发青字〔2019〕21号）精神，严格落实遴选流程，经学生自主报名、中学选拔推荐、学生网上申报、省级审核等步骤，进入到学生潜质测试环节，为了不断优化选拔机制，保证遴选过程严谨，从而遴选出具有创新潜质的中学生参与到“英才计划”中，拟统一开展学科潜质测试。现将有关事项通知如下：

    一、时间

    本次考试分为笔试和机试两部分：

    1. 潜质测试笔试考试时间：2019年12月7日9:00-10:00

    2. 潜质测试网络考试时间：2019年12月7日10:00-12:00

    二、地点

    武汉大学文理学部教二楼机房（详见附件1）

    三、参与人员

    1. 省级管理办公室相关人员

    2. 武汉大学本科生院相关人员

    3. 参与中学相关人员

    4. 2020年湖北省“英才计划”入围潜质测试全体学生

    四、注意事项

    1. 2019年12月7日8:30入考场，按学科分区域就座，准备笔试。

    2. 笔试现场将发放纸质试卷、中性笔、草稿纸等工具（计算器自备），1个小时后笔试结束交卷。随即开始网上在线测试（测试操作指南见附件2）。
