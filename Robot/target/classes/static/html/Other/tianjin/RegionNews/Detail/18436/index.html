河北区举办中小学校园文化科技节中期展示活?/p>

2012/11/5 本文被阅读过13981?/div>

    河北区首届中小学校园文化科技节中期展示活动于2012?0?0日下午在光明小学圆满召开，参加此次会议的专家、领导有中国航天工业总公司三院八三五八研究所正局级顾问、研究员、尹怀勤书记；中国宇航学会科普处钟铃处长，段丽娟老师；天津市青少年科技中心安庆红副主任、竞赛部李军部长；航天科普作家谈煦老师；河北区教育局魏薇副局长、中学科李熠科长、小学科赵文丽科长及河北区各校的科技辅导员?/span>
    本次活动共开展了学生现场制作火箭航天飞机表演?/span>参观航天文化科普展；参会领导和专家现场题词、学校领导向参会领导和专家赠送学校自编的航天校本教材，各级领导还?ldquo;光明小学航天俱乐?rdquo;揭牌仪式?/span>谈煦老师讲座“中国航天事业的发展，天津航天特色”等几项活动?/span>
      光明小学几年来开展航天活动和打造航天特色的情况，积极引进航天教育资源，拓展空间，让师生共同分享航天文化的魅力。在教育局、少年宫领导的支持和关心下，积极与航天专家联系，制定出有自己特色的航天文化课程规划。航天专家尹怀勤、清华大学高云峰教授都十分关心教材的编写，并给予了指导，特别是谈煦老师还亲自来到学校对教材进行了修订。在去年参加全国“天宫一?rdquo;科学搭载活动中，荣获全国5个唯一的搭载奖，成为天津市唯一获此殊荣的学校?/span>
      此次活动得到了与会专家、领导的首肯和好评，展示了光明小学打造航天校园文化办学特色?/span>
