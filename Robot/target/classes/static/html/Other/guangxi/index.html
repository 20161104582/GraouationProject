2019年广西中小学教师科学营活动圆满结束

11月9日，2019年广西中小学教师科学营活动在北京圆满结束。11月5-9日的这场殿堂之旅既让老师参观了国家重大科技工程项目、重点实验室、高科技院所、创新实践基地，参与了重点高校的探究性学习，又聆听了中国科学院院士、一线科技工作者和重点大、中小学名师的“传业授道解惑”。“科学技术发展的无限可能”，“国之重器与民族脊梁彰显的时代精神”，“青少年科技教育的出路何在”——为期5天的学习让200名来自广西14个地市基层中小学的科技教师陷入深深的思考，与茅塞顿开的喜悦、油然而生的家国情怀一起营造了科学营的独特氛围。

2019年广西中小学教师科学营活动在北京开营

11月5日，2019年广西中小学教师科学营活动在北京拉开序幕。活动将持续一周，设物理、化学、生物和小学四个分营。200名来自广西14个地市基层中小学的老师将深入国家重大科技工程项目、重点实验室、创新实践基地、高科技院所、重点高校和中小学校、科普场馆等参观学习，并聆听院士报告，与科学家、科技工作者和名师互动交流。

广西设立青少年科技创新自治区主席奖

近日，经中央批准，全国评比达标表彰工作协调小组函复广西壮族自治区党委、政府，同意设立“广西青少年科技创新自治区主席奖”（含主席提名奖）。

第三届“一带一路”青少年创客营与教师研讨活动开幕式在广西南宁举办

9月25日，第三届“一带一路”青少年创客营与教师研讨活动在广西南宁市举行。中国科协副主席、书记处书记孟庆海，广西壮族自治区人民政府副主席、党组成员黄俊华等领导嘉宾出席开幕式。本届活动秉承第二届“一带一路”国际合作高峰论坛会议精神，活动主题为“开放融合•创新发展”。活动于2019年9月24至30日举行，为期7天，作为第16届中国-东盟博览会的重点板块之一，也是活动首次在北京以外的城市开展，这成为了本届活动的最大亮点。

我馆与与东南亚教育部长组织科学教师与科学教育工作者进修区域中心签署会谈谅解备忘录
