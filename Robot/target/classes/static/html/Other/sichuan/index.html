第十九届“明天小小科学家”奖励活动简讯

由中国科协、中国科学院、中国工程院、国家自然科学基金委员会和周凯旋基金会共同主办的第19届“明天小小科学家”奖励活动终评于11月6-11日在中国科学院大学举行，来自全国18个省及香港、澳门特别行政区的126名高中学生入围终评。今年我省申报项目作品88件，经过初评13件项目作品入围终评。

2019年青少年高校科学营四川分营总结研讨会在成都成功召开

11月28日—29日，2019年青少年高校科学营四川分营总结研讨会在成都成功召开。四川省青少年科技中心副主任夏弘，中国科学院、水利部成都山地灾害与环境研究所科普与基地处处长张宁，四川大学、电子科技大学、中科院成都山地所三所承办单位的项目主管，以及来自全省21个市（州）的分管领导、项目主管，部分教师代表和志愿者代表共45人参加此次会议。
