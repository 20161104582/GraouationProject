天津市科技周专项活动在大港成功举办

2013/5/27 本文被阅读过13910?/div>
5?/span>20?/span>，由天津市科委、天津市教委、天津市科协、滨海新区科技周活动组委会主办，大港科技局、大港教育局、大港科协、天津市青少年科技中心、天津市科普中心承办?ldquo;青少年科技创新区域示范暨大港青少年科普成果展示活动”在大港七中隆重开幕。天津市相关单位领导、各区县代表、外国友人和大港各校师生代表?/span>800人参加了这次科技盛会。天津市科委副主席贾堤、滨海新区副区长郭景平、大港管委会副主任高相忠、大港教育局局长刘桂梧出席活动启动仪式并讲话。会上与会领导为获得天津?ldquo;科技创新操作?rdquo;的大港二中、大港七中、刘岗庄中学授牌。展示活动中，与会人员参观了大港科技发明作品展、科技成果图片展、科技教育局档案展和科幻画创新制作展，观摩了大港二中“绿地?rdquo;测绘现场展示、大港四中航模（车模）探究体验活动、大港七?ldquo;无线电测?rdquo;、刘岗庄中学“湿地观测航拍”和天津市青少年户外科技探究实践等近20项探究体验活动。活动期间还展示了天文、奇思妙想、无线电?/span>3节富有特色的科技教学课，央视大型公益活动“最受关注的乡村教师”刘月升老师还作?ldquo;让科学的种子在乡村生根开?rdquo;事迹报告?/span>
