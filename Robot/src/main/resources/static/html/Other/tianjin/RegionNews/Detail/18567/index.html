    本届大赛?ldquo;大港第六届青少年科技创新大赛”。大赛分为非现场比赛和现场比赛两大类。非现场比赛共设：学生发明、小学生创新设计方案、教师发明、青少年科技实践活动、科学DV?10个比赛项目，42所学校共上?028件作品，其中小学生创新设计方?2件作品；师生发明作品134件，比去年多?0余件；教师科教制?6件作品，比去年多?1件，经专家评审，共有373件作品获奖?/span>
    现场比赛共设越野车赛、鸡蛋撞地球、纸桥承重、小学搭高塔、简易机器人、水火箭、电子百拼、智能车八个比赛项目?7所学校，共?043名选手参加比赛。大赛期间还对师生发明、科教制作、科幻画、科学DV等优秀作品进行展示，同时展出各校科技教育活动展牌共计112块。此外，天津市市、滨海新区和大港科协的领导，以及新区、大港教育局领导莅临大会，观摩指导，对大港科技教育取得成绩和第六届青少年科技创新大赛的成功举办予以了高度评价?/span>
