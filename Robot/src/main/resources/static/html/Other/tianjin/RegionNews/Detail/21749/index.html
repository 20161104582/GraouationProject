大港教育系统举办第七届科技创新大赛

2013/12/2 本文被阅读过14721?/div>
2013?/span>11?/span>23?/span>，大港六中校园人头攒动，来自大港教育系统和社会各界的朋友1500人前来观摩和参加大港教育系统第七届青少年科技创新大赛。天津市科普协会普及部部长张新智、市青少年活动中心竞赛部部长李军，新区科技局和科协领导，以及新区教育局领导参加了当天的活动?/span>
本次大赛分为非现场比赛和现场比赛两大类。现场比赛设越野车赛、鸡蛋撞地球、纸桥承重、小学搭高塔、简易机器人、水火箭、电子百拼、智能车八个比赛项目，辖区内45所中小学校902名队员参加了现场比赛?/span>161人获奖。非现场比赛设学生发明、小学生创新设计方案、教师发明、青少年科技实践活动、科?/span>DV?/span> 10个比赛项目，辖区?/span>38所中小学校共上?/span>961件作品，经专家评审，?/span>462件作品获奖?/span>
历时两个多月的第七届青少年科技创新大赛在为期半天的现场比赛中落下帷幕，经过激烈的角逐，共有14所学校获团体奖，上古林小学、太平村三小、大港三小、大港七小、英语实验小学、大港六小获小学组前六名，刘岗庄中学、大港二中、大港六中、大港七中、太平村二中、大港九中获初中组前六名，太平村一中、大港实验中学获高中组前两名?/span>
近几年，大港教育系统秉承“科技人才的培养要从娃娃抓?rdquo;的理念，立足科普知识的推广与传播，立足常态化科技教育教学工作，积极为青少年营造爱科学、学科学、用科学的氛围，开展了丰富多彩、深受学生喜爱的校内外科学探究与实践活动。从2007年至今，已经连续七年成功举办了青少年科技创新大赛，在全系统掀起了科技创新活动的热潮，思想逐步深入人心，科技竞赛硕果累累，涌现出一大批优秀科技辅导员和具有科技发展潜力的优秀学生，在全国及市区级各类科技大赛中均取得优异成绩，逐步形成了参与广、影响大、基础实、有特色、成绩好的科技教育工作局面?/span>
