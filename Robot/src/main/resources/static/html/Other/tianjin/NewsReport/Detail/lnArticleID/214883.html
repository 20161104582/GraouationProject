3J携手，共创青少年科技教育新辉?--2017京津冀青少年机器人竞赛及青少年科技创新教育研讨会纪?/p>

2017/3/21 本文被阅读过14097?/div>

    作为2017年京津冀青少年科技教育协同发展的一部分??6?18日，来自北京、天津和河北省的青少年机器人小选手、领队和科技辅导员齐聚北京房山，就青少年机器人项目进行了展示交流?/p>

        

    在比赛期间还召开了京津冀三地青少年科技创新教育资源一体化工作座谈会。北京青少年科技中心主任张晓虎、副主任吴媛、项目科科长赵峥、天津青少年科技中心主任安庆红、副主任张恺、河北省青少年科技中心主任苑社民等就京津冀青少年科技教育协同发展和资源一体化等进行了深入交流和沟通，这也?017年三地在青少年科技教育领域的第三次沟通与研讨?/p>

        

    北京青少年科技中心主任张晓虎详细介绍了北京青少年机器人竞赛工作发展历史及本届赛事情况，以及北京市青少年科技教育的特色与资源优势，天津和河北省两地也分别就各自开展的工作、资源特色等进行了介绍与交流?/p>

    天津青少年科技中心建议就如何用好首都青少年科技教育资源，如何在突出地域特色的基础上为三地青少年科技教育做实事，以推动三地青少年科技创新教育协同发展进行深入的交流与研讨，真正发挥各自资源与优势，强强联手形成合力，促进京津冀青少年科技教育的整体提升?/p>

         

    三地青少年科技中心一致表示积极响应党中央、中国科协和天津市科协对京津冀科技教育一体化、协同发展的指示精神，三地连动、优势互补，辐射三地青少年科技教育，为其他省市青少年科技教育的发展提供经验与借鉴，会议上还初步拟定了近期的重点工作和调研方向?/p>
