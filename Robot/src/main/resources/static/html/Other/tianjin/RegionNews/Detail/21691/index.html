“弘扬航天精神实现中国梦想”——天津市第十四中学航天英雄进校园活动

2013/11/20 本文被阅读过13671?/div>

11?span lang="EN-US">15?/span>，英雄航天员聂海胜、张晓光、王亚平走进了十四中学校园，来到了我校师生中间，出席“弘扬航天精神实现中国梦想”——天津市第十四中学航天英雄进校园活动?/span>

下午四时许，全校师生在前广场和教学楼内热烈欢迎航天员们莅临我校。三位航天员接过学生献花后在校领导的陪同下走进校展厅参观了航天航空主题成果展示活动并题字留念?/font>

随后，航天员们在依次参观过学校国际交流中心、数字博物馆?span lang="EN-US">ipad交互教室?span lang="EN-US">3D打印中心后进入报告厅开始主题活动。高虎校长首先致欢迎辞，市区领导和英雄航天员聂海胜同志分别讲话。英雄航天员王亚平在十四中学“问学”讲堂上向在场广大师生再次展示了太空授课的教具和当时的真实情景，使大家增长了许多航空物理知识，了解了航天员的背后艰辛的故事?/font>

英雄航天员们还与各级领导为我?ldquo;神舟?rdquo;航天航空知识大奖赛获奖学生代表颁发奖状，与在场师生进行近距离互动交流，并向提问学生赠送了亲笔签名的《中国航天员杂志》?/font>

为了进一步表达我校师生对航天英雄最真挚的敬意，高虎校长向他们颁发了 “科技辅导?rdquo;聘书，并请他们向杨利伟同志转?ldquo;荣誉校长”聘书。之后航天员们还与我校师生互赠了礼物?/font>

最后航天员与师生共同欣赏了我校香樟之韵合唱团的精彩演出?/font>

通过与航天英雄的近距离接触，激发了我校广大师生以航天员为榜样、向航天英雄学习的情感，增强了学生的民族自豪感，提高了学生对航天知识的兴趣?/font>

相信通过这次活动，同学们能够胸怀高远抱负，以执着的信念、优良的品德、丰富的知识、过硬的本领、勇敢地担负起历史重任，让航天精神在学习和生活中放射出更加夺目的时代光芒，点亮学生心中的中国梦想?/span>
