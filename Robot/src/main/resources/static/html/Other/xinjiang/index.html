科技助力精准扶贫——2019年和田地区科技辅导员培训班成功举办

5月27日至30日，由自治区科协主办，自治区青少年科技中心、自治区青少年科技辅导员协会、和田地区科协共同承办的2019年和田地区科技辅导员培训班在和田市成功举办。来自和田地区七县一市的百名各族农村学校科技辅导员、科技教师参加了此次培训。

新疆科协助力五项学科竞赛

2019年4月17日下午，自治区科协组织召开了自治区五项学科竞赛秘书长联席会议，自治区科协副巡视员李永晟、新疆科技馆馆长战强、自治区科普活动中心主任苏红、自治区五大学科竞赛主办单位（新疆数学学会、新疆物理学会、新疆化学学会、新疆动植物学会、新疆青少年科技辅导员学会）负责同志及自治区科普活动中心相关同志出席会议。

为深入贯彻乡村振兴计划和科技助力精准扶贫计划，支持薄弱地区青少年科技教育工作，促进青少年科技教育平衡充分发展。5月27日至30日，由自治区科协主办，自治区青少年科技中心、自治区青少年科技辅导员协会、和田地区科协共同承办的2019年和田地区科技辅导员培训班在和田市成功举办。来自和田地区七县一市的百名各族农村学校科技辅导员、科技教师参加了此次培训。

培训期间，主办单位为科技辅导员们安排了充实而有丰富的培训内容。北京航空航天大学副教授秦曾昌带来的人工智能发展史科普讲座，新颖有趣的科技前沿知识，开拓了基层教师们的视野；克拉玛依市青少年科技活动中心董庆明老师带来的手掷飞机、李曦繇老师的《一张纸的使命》、魏浩然老师的《创意编程与智能硬件》等综合实践课程受到了老师们的热烈欢迎。无论是科普报告还是实践操作，大家都聚精会神，积极参与。老师们纷纷表示，在和田地区这样的欠发达地区很少有机会体验如此前沿、优质的教育资源，这次参加培训受益匪浅。他们表示一定会把这次的所学所感运用到平时的教学工作中去，为进一步提升和田地区的科技教育水平做出贡献。

开展面向贫困地区科技辅导员培训，旨在贯彻落实《全民科学素质行动计划纲要实施方案（2016-2020年）》文件精神和中国科协“科技助力精准扶贫工程”有关要求，充分发挥自治区科协自身优势，将扶贫与扶志扶智相结合，通过建立健全自治区、市、县三级科技辅导员培训体系，加大贫困地区科普资源聚集，培育科技教育的骨干会员队伍，提升贫困地区科技辅导员的专业能力和素质，促进优质科教资源的共建共享，为建设和谐稳定的新疆奠定扎实的工作基础。

       5月24日,新疆科协十七楼会议室座无虚席,150名来自全疆各地的信息技术老师,在这里参加了由新疆科普活动中心和新疆青少年科技辅导员协会主办的2019年自治区青少年创意编程与智能设计活动骨干教师培训班。

       本次培训是在互联网、大数据、人工智能等新的时代背景下，为推进科技辅导员业务能力和综合素质的提升，适应新一轮科技革命和产业变革的发展趋势，培养人工智能领域科技创新后备人才的积极探索和实践。培训邀请到网易资深项目运营专家，2018年全国创意编程与智能设计大赛、全国青少年创意编程体验活动运营负责人，网易卡搭精英教师计划发起人顾心远老师和网易卡搭商务市场总监，高级计算机工程师范星星老师，为科技辅导员们从国内外少儿编程的发展趋势和教育意义、scratch和python语言的基本介绍等方面进行了讲解，并且安排互动体验环节让科技辅导员们参与scratch作品的创作。

       学员们都表示本次培训虽然时间短，但内容饱满，讲座和动手实践扎实有用，既提升了自己的专业水平，又结交了良师益友，收获满满。他们将把学到的知识、技能以及思维方式应用到日常的教学活动之中，在培养人工智能领域科技创新后备人才方面进行积极的探索和实践。

 

                        新疆科普活动中心

                        2019年5月26日

伊犁哈萨克自治州科协、各地州市科协、各有关单位：

       2019年全国青少年创意编程与智能设计大赛拟于4——10月举办。为扎实做好我区参赛准备工作，帮助科技辅导员掌握网上参赛规程，加深对青少年人工智能科技教育活动的理解，开阔科技辅导员的视野，进而向广大青少年普及推广人工智能相关科普知识，提高他们对人工智能的认知和初步应用能力，自治区科协决定举办自治区青少年创意编程与智能设计骨干教师培训班，现将有关事项通知如下：

       一、组织单位

      主办单位:自治区科协

      承办单位:自治区科普活动中心（自治区青少年科技中心）、自治区青少年科技辅导员协会

      二、培训时间和地点

      2019年5月24日在自治区科协17楼会议室参加培训（5月23日在福建大酒店一楼报到，25日离训）。

      三、参训人员要求

      具备相关专业学习背景，有三年以上一线教育教学经验，主要承担科技类综合实践活动课程（包括信息技术、劳动技术、通用技术、科学等）的教学工作。有两年以上任意编程语言教学经验，熟悉开源硬件开发，并有意愿开展教学或社团活动的教师。

      四、其它事项

      1.参加培训人员食宿由会务统一安排，住宿费及往返交通费由派出单位承担，餐费由会务承担。

      2.参加培训人员自带笔记本电脑，自行下载安装编程软件和工具。（电脑操作系统为WIN7及以上版本，可提前登陆http://www.kada.163.com熟悉平台，培训当天统一进行注册。）

      3.请报名参加培训的学员将《培训回执》(见附件)于5月15日前发送至指定邮箱。
