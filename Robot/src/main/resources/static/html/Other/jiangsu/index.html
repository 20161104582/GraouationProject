2019—2020年度江苏省全民科学素质大赛

为深入实施《江苏省全民科学素质行动计划纲要实施方案（2016—2020年）》，大力开展常态化、制度化和社会化科普活动，在全社会积极营造浓郁的科学文化氛围，促进我省全民科学素质水平不断提升，2019年继续组织开展“江苏省全民科学素质大赛”。活动时间:2019年9月7日至12月31日

江苏选手获“明天小小科学家”称号

11月11日，由中国科协、中国科学院、中国工程院、国家自然科学基金委和周凯旋基金会共同主办的第19届“明天小小科学家”奖励活动在中国科学院大学圆满落幕，江苏选手取得骄人成绩。江苏英才计划学员、省苏州中学的卢冠宇同学以第一名的成绩，荣获“明天小小科学家”称号，全国仅有3名同学获此殊荣。

2019中国青少年科技辅导员协会年会暨江苏省青少年科技教育高峰论坛在南京举行

11月1-2日，2019中国青少年科技辅导员协会年会在南京圆满举行。本次年会在中国科协科普部和中国科协青少年科技中心支持下，由中国青少年科技辅导员协会、江苏省科学技术协会联合主办，江苏省青少年科技教育协会、江苏省青少年科技中心、江苏省学会服务中心承办，南京大学、东南大学、南京航空航天大学、南京理工大学、南京师范大学、江苏省人工智能学会、江苏省工程师学会、江苏省航空航天学会、南京青少年科技教育协会共同协办。围绕“科教联合、共育英才”主题，吸引了全国各地1000余名科技教育专家、优秀科技辅导员、中小学校长参加，搭建教育与科技沟通的桥梁，凝心聚力，交流研讨如何将科技界与教育界的智慧和力量汇聚，多视角、多维度分析解读新时代科技教育，共同推进新时代青少年科技教育创新发展。

第五届中国“互联网+”大学生创新创业大赛圆满落幕江苏队勇创佳绩

10月15日晚，第五届中国“互联网+”大学生创新创业大赛颁奖仪式在浙江大学落下帷幕，教育部副部长钟登华出席颁奖仪式。大赛共评选出61个主赛道金奖项目、15个红旅赛道金奖项目、15个职教赛道金奖项目以及20个萌芽赛道创新奖项目。经过激烈角逐，最终江苏代表队各赛道获奖总数位居全国第一，特别是在大赛新设立的萌芽版块中，我省中学生斩获5枚创新奖，以绝对优势位居全国第一。

  • 《STEM项目学生研究手册》

    《STEM项目学生研究手册》

    《STEM项目学生研究手册》是一本针对高中学生和教师的全面而详实的学习资料。它帮助教师在科学研究过程中的不同阶段对学生进行指导，并对相应的教学步骤提供了详尽的描述，甚至如果老师从没有科学实验设计经验，也能够在本书的帮助下指导学生进行科学、技术、工程学和数学实验研究。书中“研究设计、背景研究、假设的撰写、研究方案的撰写”等章节可以有效地帮助学生构思和完成他们的研究项目。

  • 世界珍禽——朱鹮

    世界珍禽——朱鹮

    陕西汉中朱鹮国家级自然保护区位于陕西秦岭南坡，朱鹮，当今世界最濒危的鸟类之一，被世界自然保护联盟（IUCN）列入极濒危物种名录，国家Ⅰ级保护动物。历史上曾广 泛分布于中国、日本、朝鲜半岛和俄罗斯东部。二十世纪初在世界范围内朱鹮的分布和数量逐渐减少，二十年代人们认为日本的朱鹮已不存在，但 后来又发现少量残存于佐渡和能登半岛的个体。
