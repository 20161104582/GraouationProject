第十四届中国青少年机器人竞赛

第十四届机器人竞赛：我们为梦想而战
青少年机器人爱好者天山脚下体验“激情创造快乐成长”
第十四届中国青少年机器人竞赛在乌鲁木齐开赛
第十四届中国青少年机器人竞赛机器人入场引围观

第十四届中国青少年机器人竞赛于2014年7月15日至19日新疆自治区乌鲁木齐市举行，竞赛主题为“激情创造，快乐成长”。包括机器人创意比赛、机器人综合技能比赛、机器人足球比赛、机器人FLL工程挑战赛和机器人VEX工程挑战赛五项分赛。共有26个省（自治区、直辖市）和新疆建设兵团在内的222支参赛队，共计594名队员和232名教练员参加。

精彩赛事

机器人综合技能比赛

机器人综合技能比赛是中国青少年机器人竞赛项目之一。其活动对象为中小学生，要求参加比赛的代表队在现场自行拼装机器人、编制机器人运行程序、调试和操作机器人。参赛的机器人是程序控制的，可以在赛前公布的竞赛场地上，按照本规则进行比赛活动。
　　本届机器人综合技能比赛的主题为“少年当自强”。中国美好的明天在我们手中！实现中华民族复兴的中国梦是我们青少年的历史责任！让我们想想，我们和我们的机器人伙伴能为此做些什么。

机器人足球比赛

机器人足球比赛采用2对2机器人比赛规则，每支参赛队有队员2名参赛队员，使用使用自行拼装、编制运行程序的2个机器人进行比赛。比赛强调机器人足球比赛中的技术成份，而不是一味比拼速度和力量。
　　四年一届的2014年巴西世界杯已落幕，一年一届的中国青少年机器人竞赛足球世界杯正在新疆上演。哪支队伍问鼎冠军，请你猜猜看。

FLL机器人挑战赛

FLL机器人挑战赛是一项青少年国际机器人比赛项目。要求参加比赛的代表队自行设计、制作机器人并进行编程。参赛的机器人可在特定的竞赛场地上，按照规则进行比赛。激发青少年对机器人技术的兴趣，为国际FLL挑战赛选拔参赛队。
　　本届 FLL 挑战赛的主题为“ 天降之咎 ”在“天降之咎”挑战赛中，你们将研究风暴、地震、海啸及更多的自然灾害。参赛队将认识到人们生活、工作和休闲的地方发生强烈的自然事件时该做什么。在“天降之咎”挑战赛中行动起来吧！

机器人创意比赛

机器人创意比赛是基于每年一度的中国青少年机器人竞赛的主题与规 则，组织在校中小学生机器人爱好者，花费6个月左右的时间，在课题导 师或教练员的指导下，在学校、家庭、校外机器人工作室或科技实验室里， 以个人或小组的方式，进行智能机器人的创意、设计、编程与制作，最后 以具体的机器人创意作品的形式参加中国青少年机器人竞赛组委会举办的 机器人创意比赛活动。
